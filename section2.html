CSS

CSS란 무엇인가?


CSS(Cascading Style Sheets)는 HTML로 작성된 웹 문서의 스타일(디자인)을 지정하는 언어입니다. CSS를 사용하면 웹 페이지의 레이아웃, 색상, 글꼴, 간격 등 시각적 요소를 제어할 수 있습니다. CSS는 HTML 콘텐츠와 스타일을 분리함으로써 웹 페이지의 유지보수를 쉽게 하고, 사용자 경험을 향상시키는 데 중요한 역할을 합니다.


CSS의 주요 개념


1. 선택자(Selector)


선택자는 CSS에서 스타일을 적용할 HTML 요소를 지정합니다. 기본적으로 선택자는 아래와 같은 종류가 있습니다:


  • 태그 선택자: 특정 태그를 선택합니다. p { color: red; }

  • 클래스 선택자: 클래스 이름으로 요소를 선택합니다. .box { border: 1px solid black; }

  • ID 선택자: ID 이름으로 요소를 선택합니다. #header { background-color: blue; }

  • 그룹 선택자: 여러 선택자에 동일한 스타일을 적용합니다. h1, h2, h3 { font-weight: bold; }


2. 속성(Property)와 값(Value)


CSS는 속성과 값의 쌍으로 구성됩니다. 속성은 스타일을 변경하려는 요소의 특성을, 값은 그 속성에 적용할 구체적인 스타일을 나타냅니다. 

h1 {
                  color: red; /* 글자 색상을 빨간색으로 */
                  font-size: 24px; /* 글자 크기를 24px로 */
                }

CSS 적용 방법


CSS는 HTML 문서에 세 가지 방법으로 적용할 수 있습니다:


1. 인라인 스타일


HTML 요소의 style 속성을 사용하여 CSS를 적용합니다.

<p style="color: blue; text-align: center;">이 문장은 파란색으로 가운데 정렬됩니다.</p>

2. 내부 스타일 시트


HTML 문서의 <head> 태그 안에 <style> 태그를 사용하여 CSS를 정의합니다. 

<style>
                  p {
                    color: green;
                    font-size: 18px;
                  }
                </style>

3. 외부 스타일 시트


별도의 .css 파일에 CSS 코드를 작성하고 HTML 문서에서 <link> 태그로 연결합니다.

/* styles.css */
                p {
                  color: black;
                  line-height: 1.6;
                }
<link rel="stylesheet" href="styles.css">

자주 사용하는 CSS 속성


1. 색상 및 배경



  • color: 텍스트 색상을 지정합니다. color: red;

  • background-color: 요소의 배경색을 지정합니다. background-color: yellow;

  • background-image: 배경 이미지를 설정합니다. background-image: url('image.jpg');


2. 글꼴



  • font-family: 글꼴을 지정합니다. font-family: Arial, sans-serif;

  • font-size: 글자 크기를 지정합니다. font-size: 16px;

  • font-weight: 글자 굵기를 지정합니다. font-weight: bold;


3. 레이아웃



  • margin: 요소의 외부 여백을 설정합니다. margin: 20px;

  • padding: 요소의 내부 여백을 설정합니다. padding: 10px;

  • border: 요소의 테두리를 설정합니다. border: 1px solid black;


4. 디스플레이 및 배치



  • display: 요소의 표시 방식을 설정합니다. display: block;, display: inline;, display: flex;

  • position: 요소의 위치를 설정합니다. position: relative;, position: absolute;

  • z-index: 요소의 쌓임 순서를 설정합니다. z-index: 10;


헷갈릴 수 있는 개념들



  • margin vs padding:
    • margin: 요소와 요소 사이의 외부 여백
    • padding: 요소 내부 콘텐츠와 테두리 사이의 여백

  • inline vs block:
    • inline: 요소가 한 줄에 배치되며 크기 조정이 제한적입니다.
    • block: 요소가 전체 너비를 차지하며 줄바꿈이 일어납니다.

  • id vs class:
    • id: 고유 식별자를 제공하며 한 문서에 한 번만 사용
    • class: 여러 요소에 적용 가능하며 재사용 가능
